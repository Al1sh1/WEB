🎧 About Spotify Discovery

Project Overview

This Angular application provides a premium discovery experience for Spotify's top artists. It was migrated from an e-commerce and anime theme to a sleek, modern music interface that integrates with song data and high-quality artist galleries.

Core Features

  • 🎤 Discovery of the top 100 global Spotify artists
  • 🎵 Top Track galleries for every artist with high-res covers
  • 🖼️ Interactive photo grids with play-button micro-interactions
  • ✏️ Artist management (Edit artist names locally)
  • 🗑️ Dynamic list management with UI updates
  • ✨ Spotify Dark Theme with signature "Circular" aesthetics
